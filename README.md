Highland Games - Full-stack demo (mobile-first)
================================================

This project is a proof-of-concept for the coursework:
- React (Vite) + Tailwind on the client (mobile-first)
- Node.js + Express + MongoDB on the server (auth implemented)
- Only login/register interact with the database as requested

To run locally:

1. Client:
   cd client
   npm install
   npm run dev

2. Server:
   cd server
   npm install
   copy .env.example to .env and provide MONGO_URI and JWT_SECRET
   npm run dev
